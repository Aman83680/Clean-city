🌿 Clean City - Smart Cleanliness Reporting System

Clean City is a modern web application designed to encourage citizen participation in maintaining a cleaner environment. Users can report cleanliness issues, earn reward points for contributions, and compete on a community leaderboard.

✨ Features
📢 Report cleanliness issues
📍 Location-based issue reporting
🏆 Reward points system
📊 Dynamic leaderboard
💾 Local Storage data persistence
📱 Responsive design
🌱 Clean and modern user interface
🏅 Community engagement through gamification
🛠️ Tech Stack
HTML5
CSS3
JavaScript
Local Storage
Future Enhancements
User Authentication
GPS Location Detection
Image Upload & Preview
Admin Dashboard
Issue Status Tracking
Firebase Integration
Objective

To promote civic responsibility and encourage citizens to actively participate in keeping their cities clean by providing a simple and engaging reporting platform.
